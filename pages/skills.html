SKILLS

A lo largo de mi carrera profesional, he adquirido un profundo conocimiento en herramientas y tecnologías que son clave para el análisis de datos, la automatización de procesos y el desarrollo de soluciones innovadoras. Entre mis habilidades más destacadas se encuentra el manejo avanzado de Excel, donde aplico técnicas como la creación de tablas dinámicas y generación de reportes para la toma de decisiones estratégicas.

Complementariamente, domino Power BI, lo que me permite diseñar dashboards interactivos y realizar modelados de datos avanzados que impulsan el análisis empresarial.

En cuanto al manejo de bases de datos, mi experiencia incluye tanto sistemas relacionales como MySQL, donde diseño estructuras optimizadas y consultas eficientes, así como también, en bases no relacionales como MongoDB, adecuadas para trabajar con datos no estructurados y escalables.También estudié Python, creando scripts personalizados para el análisis de datos, automatización de tareas y visualización de información.

En proyectos de desarrollo, utilizo Git y GitHub para el control de versiones y la colaboración en equipo, asegurando la trazabilidad del código. Mi conocimiento de SAP se centra especialmente en conciliaciones y reportes financieros, mientras que mi habilidad con Visual Studio Code me permite optimizar proyectos mediante un entorno de desarrollo versátil y eficiente.
